    Spike Valorant

    Modelo a escala del videojuego Valorant usado en los modos de juego en los que los defensores deben desactivarla y los atacantes plantarla. El módulo usado es un ESP32 programado con el IDE de arduino, es posible activar y desactivar desde una app en tu telefono móvil.

    Reposamuñecas

    Reposamuñecas funcional, con el que puedes iniciar sesión en Windows mediante bluetooth (BLE(bluetooth low energy)). Los módulos usados son: ESP32, RFID RC522 y pantalla OLED SDD1306 alimentado con cable USB-C.

    Control de turnos

    Aplicación de gestión de agendas que contiene secciones Médico - Paciente - Especialidad. Podemos asignar dia y hora a un Paciente y relacionarlo con su Médico, de igual manera se le puede asignar una Especialidad a éste ultimo (cirujano, dentista, dermatólogo ...). Base de datos creada en SQL Server y plugin FullCalendar con Jquery y Ajax. Seguridad implementada con AntiForgery Token.
